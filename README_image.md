## 一、第3章 — 灵敏度分析

### Fig 3-1：SPM参数Sobol灵敏度分析

| 项目               | 说明                                                                            |
| ------------------ | ------------------------------------------------------------------------------- |
| **文件**     | `plots/sensitivity/第3章_Fig3-1_灵敏度分析.png`                               |
| **来源代码** | `src/sensitivity.py` → `SensitivityAnalyzer.plot_sensitivity()`            |
| **数据来源** | B0005 第0次循环的放电数据（下采样至100个数据点）                                |
| **分析方法** | Sobol全局灵敏度分析（Saltelli采样，样本数=256，共256×(2×8+2)=4608次仿真评估） |

**图表内容**：

柱状图展示8个SPM参数对电压RMSE输出的**一阶灵敏度指数(S1)**和**总效应灵敏度指数(ST)**。参数包括：

- 正/负极扩散系数（$D_p$, $D_n$）
- 正/负极反应速率常数（$k_p$, $k_n$）
- 正/负极颗粒半径（$R_p$, $R_n$）
- 正/负极最大浓度（$c_{max,p}$, $c_{max,n}$）

> **注**：$R_f$（欧姆内阻）可通过EIS直接测量，不作为待辨识参数，因此不纳入灵敏度分析。

**结果与意义**：

- S1（一阶指数）反映各参数**独立**对输出方差的贡献比例
- ST（总效应指数）反映各参数独立贡献**加上与其他参数交互作用**的总贡献
- 该分析揭示了哪些SPM参数对电压预测精度影响最大，为后续参数辨识提供重点关注方向
- 灵敏度较低的参数可以适当简化处理，提高辨识效率

## 五、第7章 — 鲁棒性分析

### Fig 7-1：传感器噪声鲁棒性分析

| 项目               | 说明                                                           |
| ------------------ | -------------------------------------------------------------- |
| **文件**     | `plots/robustness/第7章_Fig7-1_鲁棒性分析.png`               |
| **来源代码** | `src/robustness.py` → `RobustnessAnalyzer.plot_results()` |
| **数据来源** | B0018在不同噪声水平下的SP-LSTM预测RMSE                         |
| **实验设置** | 噪声水平 0%/1%/2%/3%，每级重复5次取平均                        |

**图表内容**：

折线图展示不同传感器噪声水平（%）下SP-LSTM的SOH估计RMSE变化，每个数据点标注精确数值。

**实验结果**：

| 噪声水平 | RMSE    |
| -------- | ------- |
| 0%       | 0.01418 |
| 1%       | 0.12993 |
| 2%       | 0.16466 |
| 3%       | 0.16809 |

**噪声注入方法**：

仅对电压测量信号施加高斯白噪声：$V_{noisy} = V + \mathcal{N}(0, \sigma^2)$，其中 $\sigma = \text{noise\_level} \times (V_{max} - V_{min})$。电流不加噪声（电流是充放电仪的控制量，精度远高于电压传感器）。SPM仿真使用干净电流驱动，保证物理模型稳定性。噪声注入后经过中值滤波去噪预处理（窗口大小=5），然后重新进行SPM参数辨识和特征提取，评估整个pipeline的端到端鲁棒性。

**意义**：

- 评估SP-LSTM框架在实际传感器噪声环境下的可靠性
- 0%噪声下RMSE=0.01418（与无噪声训练结果一致），验证了分析流程的正确性
- 随噪声增加RMSE上升明显，说明SPM参数辨识对输入数据质量敏感
- 为实际工程应用提供了噪声容限参考，建议传感器精度控制在1%以内
